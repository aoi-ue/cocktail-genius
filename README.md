# Instructions
## Getting started

- Fork this repo
- `git clone <repo-url>` 
- `cd` into directory
- Run 'npm install' 
- Start Application, Enjoy! 

## Getting around 

This is a Web App to generate a cocktail from your choice of liquor. Enjoy! 

Demo: https://cocktail-genius.netlify.com/
